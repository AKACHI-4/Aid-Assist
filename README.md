## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AKACHI-4/Aid-Employ.git
   ```
2. Install dependencies:
   ```bash
   cd Aid-Employ
   cd app
   flutter pub get
   ```
3. Run the app:
   ```bash
   flutter run
   ```
